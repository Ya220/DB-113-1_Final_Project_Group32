# DB-113-1_Final_Project_Group32
資料庫管理（113-1）期末專案 Group 32
## 專案簡介
我們這一組的主題是二手書交易平台，此平台希望為使用者提供一個便捷的二手書買賣環境，讓使用者能輕鬆進行書籍資訊的上傳、查詢以及交易紀錄的管理。

## 安裝說明
1.環境需求
- Python 3.8 或以上
- PostgreSQL 12 或以上
- 需要安裝套件 flask、flask_cors、psycopg2、datetime  
2.安裝步驟
- 下載專案資料
- 安裝需要的套件、環境
- 設置資料庫
- 更改 app.py 跟資料庫的連線資料
- 執行 app.py
- 打開 inidex.html 即可開始使用
